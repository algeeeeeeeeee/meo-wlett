# 🐱 Meowlett — Purr-fect finances

Finance tracker app built with **Vite + React**. All data stored locally on your device (no backend needed).

## Features
- 📊 Dashboard dengan overview pengeluaran & weekly trend
- 💸 Catat transaksi dengan kategori
- 📈 Laporan & analytics bulanan
- 💕 Date budget tracker & wishlist
- 🎯 Savings goals
- 🔄 Recurring transactions
- 💳 Cicilan tracker
- 🌙 Dark mode + 6 color themes
- 🇮🇩🇺🇸 Bilingual (Bahasa Indonesia / English)
- 💾 Backup & restore (JSON)
- 📱 PWA — add to home screen

## Tech Stack
- Vite + React (JSX)
- Lucide React icons
- localStorage + IndexedDB (no backend)

## Getting Started

```bash
npm install
npm run dev
```

## Deploy ke Cloudflare Pages

Build settings di Cloudflare:
- **Framework**: Vite
- **Build command**: `npm run build`
- **Output directory**: `dist`
- **Node version**: `18`
